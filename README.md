# Profile Card Component (React)

A small React project built to practice **components and props**.
The app displays multiple user profile cards by passing data from a parent component and rendering them dynamically.

## What I Learned

* Creating reusable **React components**
* Passing data using **props**
* **Props destructuring**
* Rendering multiple components using **array + map()**
* Basic component structure in React

## Features

* Displays user profile cards
* Each card shows:

  * Profile image
  * Username
  * Role
  * Location
* Buttons for **View Profile** and **Follow**

## Tech Used

* React
* JavaScript (ES6)
* CSS

## Purpose

This project was built as part of my **React learning journey** to better understand how **data flows between components**.

## Getting Started

Clone the repository:

```
git clone https://github.com/YOUR-USERNAME/Profile-Card-Component.git
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```
